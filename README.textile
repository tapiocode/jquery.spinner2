h1. jquery.spinner2

Component for selecting numbers by typing or clicking

Original spinner can be seen here: "reaktor/jquery.spinner":http://github.com/reaktor/jquery.spinner

"Give it a spin":http://tapiocode.github.io/jquery.spinner2/ 

h1. Features

* Select amout by clicking + and - buttons
* Select amout by entering number
* Select amount by pressing up/down arrows
* Validate invalid inputs and replace by last valid value after a delay
* Trigger update event when value changes
* Option for default and min value
* Unobtrusive

h1. Additional Features in Version 2

* Support for high-density displays (retina)
* Set initial value and step-size per tag using data-attributes
* Uses Grunt
* JS-files run through jshint
* Minified css, js for production
* Max value
* Override defaults with options object, then data-attributes
