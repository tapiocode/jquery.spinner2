jQuery Spinner 2


	Usage:

	Tag(s):
		<input [class="myClass"] [data-step="1"] [data-initvalue="1"] />
	
	Script:
		$(<selector>).spinner(); // Eg. tag(s) above $('.myClass')

	Optional data-attributes:

		data-step="<integer>"       -- Value of increment/decrement
		data-initvalue="<integer>"  -- Initial value
