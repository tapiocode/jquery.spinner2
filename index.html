jQuery Spinner 2


	Usage:

	Tag(s) with attributes (see options below):
		<input 
			[class="myClass"]      // Optional class (used by selector)
			[data-step="1"]        // Optional
			[data-initvalue="1"]   // Optional
			[data-minvalue="0"]    // Optional
			[data-maxvalue="99"]   // Optional
			[data-maxlength="3"]   // Optional
			/>
	
	Script:
		$(<selector>).spinner([options]);
		// Eg. tag(s) above $('.myClass').spinner();

	Options with default values:
		{
			step:       1,  // Step size
			initvalue:  0,  // Initial value
			minvalue:   0,  // Min value
			maxvalue: 100,  // Max value
			maxlength:  3,  // Max number of digits in the input box
		}
